# Sources

This directory contains source material used to build and revise the skills in this repository.

The contents are for reference and guidance only. They are not active skills, and agents should not treat them as the current operating standard when a migrated skill exists.

Agents must not delete this directory or its contents unilaterally. Only the user may decide that source material is no longer needed and request its deletion.

## Contents

- `sources/project-references/`: archived project reference material cloned from `kibocha-solutions/project-references`.
- `reports/skills-repository-reference.md`: research report guiding the structure and migration strategy for this skills repository.
